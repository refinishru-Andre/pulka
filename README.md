# Пулька

Веб-приложение для автоматической записи и расчёта результатов очной игры в преферанс (Питер / ленинградка с домашними конвенциями).

**Live:** https://refinishru-andre.github.io/pulka/

## Что делает

Заменяет бумажный листок пули. После каждой сдачи вводишь что произошло — приложение сразу считает пулю, гору, висты каждого игрока и показывает актуальный итог «кто кому сколько должен» в вистах.

Поддерживает:
- Обычные игры 6–10 (пики / трефы / бубны / черви / БК)
- Мизер (в том числе без прикупа)
- Распасы с автоматической эскалацией 1 → 2 → 8-мерные
- «Первая рука остаётся» на 8-мерных при ремизе 8+ или пойманного мизера
- Уход без 3 (гора × 3, вистов нет)
- Сталинград (6♠ — оба вистуют автоматически)
- Полвиста на 6 и 7

## Стек

Vite + React + TypeScript + Tailwind + Zustand + PWA. Хранение — LocalStorage. Хостинг — GitHub Pages.

## Разработка

```bash
npm install
npm run dev        # http://localhost:5173
npm test           # тесты движка (26 тестов)
npm run build      # прод-сборка в dist/
```
